# commit-code 快速入门

## 简介

commit-code 是一个智能 Git 提交助手，能够分析您的代码更改并自动生成规范的提交信息。

## 快速开始

1. **准备更改**
   ```bash
   # 编辑代码文件
   # 然后暂存更改
   git add .
   ```

2. **运行技能**
   ```
   /commit-code
   ```

3. **查看分析结果并确认提交**

## 主要功能

- 🔍 **自动分析** - 识别代码更改类型和影响范围
- 🧠 **智能建议** - 生成符合规范的提交信息
- 🛡️ **安全扫描** - 检测敏感信息避免泄露
- 📊 **详细统计** - 提供代码变动统计数据

## 使用场景

- 日常代码提交
- 团队协作中的规范化提交
- 遵循 Conventional Commits 规范的项目
- 需要安全保障的代码提交

## 即时帮助

输入 `/commit-code` 即可启动技能，系统将引导您完成整个提交过程。
